# Extension Icons

These placeholder files need to be replaced with actual PNG icons.

## Required Sizes

- **icon-16.png** - 16x16px (toolbar icon)
- **icon-32.png** - 32x32px (Windows)
- **icon-48.png** - 48x48px (extension management)
- **icon-128.png** - 128x128px (Chrome Web Store)

## Design Guidelines

### Brand Identity
- Use DAN brand colors (navy blue + amber/orange gradient)
- Keep it simple and recognizable at small sizes
- Consider using a stylized "D" or book/study icon

### Suggested Concepts
1. **Letter D** with gradient fill
2. **Brain + Book** combination
3. **Flame** (representing streak) + book
4. **Graph/Chart** showing upward trend
5. **Stopwatch** with study theme

### Tools for Creating Icons
- Figma (recommended)
- Adobe Illustrator
- Canva
- Inkscape (free)
- Or commission from Fiverr/Upwork

### Export Settings
- Format: PNG
- Background: Transparent (recommended) or solid color
- Resolution: 72 DPI for web
- Color mode: RGB
